Podcasts aus RSS importieren

Importiere bestehende Folgen aus einem externen RSS-Feed. Für Datenfelder, die hier nicht unterstützt werden, wird die Kategorie automatisch auf Archiv gesetzt.

Die URL muss mit http:// oder https:// beginnen und auf einen gültigen Podcast-RSS-Feed zeigen. Alle importierten Folgen erhalten diese Kategorie, z.B. „Archiv“.
